# 브랜드 톤앤매너 가이드라인
## Lisk About 스타일 기반 홈페이지 리디자인

### 1. 브랜드 아이덴티티
**핵심 가치**: 신뢰감, 전문성, 글로벌 비전
**타겟 오디언스**: EM(Emerging Markets) 창업가, 투자자, 개발자
**브랜드 포지셔닝**: 로컬에서 시작하여 글로벌로 확장하는 성장 플랫폼

### 2. 비주얼 디자인 시스템

#### 2.1 컬러 팔레트
- **프라이머리**: 블랙 (#000000) - 주요 타이포그래피
- **세컨더리**: 화이트 (#FFFFFF) - 카드 배경, 버튼
- **액센트**: 파스텔 그라디언트 (그린-핑크-옐로우) - 배경, 하이라이트
- **텍스트**: 미드 그레이 (#666666) - 본문 텍스트

#### 2.2 타이포그래피
- **헤드라인**: 대형, 굵은 산세리프 (几何적 느낌)
- **본문**: 중간 크기, 일반 굵기 산세리프
- **버튼**: 작은 크기, 명확한 가독성

#### 2.3 UI 컴포넌트
- **버튼 스타일**: 필 형태 (rounded pill), 미니멀한 디자인
- **카드**: 화이트 배경, 부드러운 그림자
- **아이콘**: 단색 라인 아이콘, 미니멀한 스타일

### 3. 문체 가이드라인

#### 3.1 홈페이지 카피라이팅
**헤드라인**: "로컬에서 시작하여 글로벌로 확장하는 성장 플랫폼"
**서브헤드라인**: "아프리카, 동남아, 라틴아메리카에서 로컬 프로그램과 펀드를 결합하여 당신의 비전을 현실로 만들어드립니다"

#### 3.2 CTA 버튼 텍스트
- **Primary**: "인큐베이터 참여하기"
- **Secondary**: "펀딩 신청하기"
- **Navigation**: "포털 접속"

#### 3.3 네비게이션 라벨
- 인큐베이터
- 펀드
- 체인
- 앱
- 토큰
- 소개
- 블로그

### 4. 페이지별 톤앤매너 적용

#### 4.1 메인 페이지
```
시각적 요소:
- 부드러운 파스텔 그라디언트 배경
- 화이트 카드 위 블랙 타이포그래피
- 다이아몬드 프레임의 팀 사진
- 미니멀한 아이콘과 버튼

카피라이팅:
- "코드는 불을 지피고, 사람들은 변화를 만들어냅니다"
- 간결하고 신뢰감 있는 메시지 전달
```

#### 4.2 서브 페이지
```
일관된 디자인 시스템:
- 동일한 컬러 팔레트 적용
- 일관된 타이포그래피 스케일
- 통일된 버튼 스타일
- 유사한 레이아웃 구조
```

### 5. 문화적 고려사항
- 한국어 사용자를 위한 자연스러운 번역
- 글로벌 비전에 맞는 보편적인 메시지
- 기술적 용어의 명확한 전달
- 신뢰감 있는 전문가적 톤 유지

### 6. 구현 가이드
- 데스크톱 퍼스트 접근법
- 반응형 디자인 고려
- 접근성 기준 준수
- 성능 최적화 고려
